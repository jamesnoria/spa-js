# Rick & Morty API-SPA

A simple SPA-JS where I use the Rick & Morty API.


## Run Locally

Clone the project

Go to the project directory

```bash
  cd spa-js
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
